⚠️ A New Paradigm — But Not Without Pitfalls

Using AI Right Is a Critical Skill

Everyone can type a prompt. Very few understand what's actually happening — or how to use AI agents reliably and responsibly.

⚡ Known LLM Failure Modes
🌀 Hallucination
LLMs generate fluent, confident text — even when it's factually wrong. A model citing a paper that doesn't exist, or a drug dose that kills. Knowing when not to trust output is essential.
📉 Context Rot
Performance degrades as context windows grow long. Early instructions get "forgotten," reasoning drifts, and errors compound across multi-step agent tasks — a silent killer in research pipelines.
⚖️ Ethical Implications
Bias in training data, opaque decision-making, data privacy, and the risk of automating consequential decisions without oversight. Life sciences stakes are high — patients, policy, publication.
🤖 Agent Engineering — A New Craft
Writing a prompt is to AI agents what writing one line of code is to software engineering. Real agentic systems require tool design, memory management, error handling, evaluation loops, and knowing when to let the agent run — and when to stop it.
The Agent Loop (e.g. the Ralph Loop)
🧠 LLM reasons
→
🔧 calls tool
→
📋 observes result
→
✅ or stops
Each iteration updates the agent's context. Poorly designed loops accumulate errors, waste tokens, or loop forever. Knowing how to architect, debug and constrain this loop is the real engineering skill.
DDLS teaches this. You'll build agents, diagnose failure modes, and develop the critical judgment to verify what AI produces — not just prompt it.
🔬 The Changing Nature of Science

How Biological Research Has Evolved

The scientific method itself is being transformed — from purely human-driven to AI-autonomous.

Traditional
🧫
Pre-2000s → ongoing
Hypothesis-Driven Research
A scientist forms a hypothesis, designs a targeted experiment, and interprets results manually. Powerful — but slow and bounded by human capacity.
Observe → Hypothesize → Experiment → Publish
e.g. Koch's postulates, Mendel's genetics
Computational Biology
💻
2000s → present
Data-Driven Hypothesis Generation
Large-scale biological data (genomes, proteomes, single-cell profiles) analyzed computationally. ML reveals patterns that generate hypotheses no human would find.
Dataset → ML patterns → Hypothesis → Validate
e.g. GWAS, scRNA-seq, AlphaFold
🚀 Emerging Now
🤖
2024 → future
The AI Scientist — Autonomous Research
AI agents autonomously review literature, generate hypotheses, design & simulate experiments, analyze results, and draft papers. Humans set goals; AI drives the loop.
AI reviews literature → hypothesizes → iterates
e.g. Sakana "AI Scientist", MIT antibiotic AI
DDLS trains you for Stage 2 and Stage 3. Data-driven analysis, AI agent workflows, and the critical judgment to evaluate what autonomous systems produce.
